[Gym Membership Platform]

Developed with my team a platform allowing gyms to offer pay-per-minute memberships through a mobile app created using flutter and an administration website, Created using React JS. Key Features: QR scanner for session tracking, coin-based payment system, online and offline payment methods, and administration features for gyms.
